# @vp-tw/nanostores-qs

A reactive querystring manager using nanostores.

## Installation

```bash
pnpm install @vp-tw/nanostores-qs
```

## Usage

```ts
import { createQS } from "@vp-tw/nanostores-qs";

const qs = createQS(/* Options */);
```
